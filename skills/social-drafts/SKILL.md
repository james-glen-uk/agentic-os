# Social Drafts

Turn a topic (typed by hand or passed from a News Oracle card) into a
platform-tailored set of social media post drafts.

## When to use

- A trending topic is worth commenting on across social channels
- An article or artifact needs distribution content

## Process

1. Extract the single most interesting angle from the topic and headlines.
2. Draft for each platform, respecting its conventions:
   - **X/Twitter**: a 3–5 post thread; first post must hook in ≤ 200
     characters; no hashtag spam (max 2).
   - **LinkedIn**: one 120–200 word post; professional but human tone;
     line breaks for scannability; end with a question to invite comments.
3. Include the most relevant source link once per platform.
4. Vary the wording between platforms — never copy-paste the same text.

## Output

Markdown with an `## X/Twitter` section and a `## LinkedIn` section.
**Drafts only — never post anywhere.** Output is saved to the Artifact
Library automatically.

Primary: gemini
